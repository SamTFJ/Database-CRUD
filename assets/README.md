### First create the virtual .venv folder
```
python3 -m venv .venv
```

### Secondly, enter the .venv
```
source .venv/bin/activate 
```

### Then, install the requirements for it
```
.venv/bin/python3 -m pip install -r requirements.txt
```

### Finaly, to run
```
.venv/bin/python3 -m dbconnection
```
